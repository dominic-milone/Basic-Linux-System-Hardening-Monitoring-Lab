# Basic-Linux-System-Hardening-Monitoring-Lab
Secure an Ubuntu virtual machine (VM) by configuring user permissions, firewalls, and monitoring tools, and then document system activity.

Secured an Ubuntu virtual machine by updating system packages, creating and restricting user accounts, configuring a firewall (UFW), and monitoring system activity. Verified user restrictions through failed login attempts, enforced network security by allowing only SSH traffic, and observed system performance and processes using htop. Documented authentication logs and firewall behavior, demonstrating practical skills in Linux system hardening, cybersecurity, and IT operations.


Skills Demonstrated:

Linux user management

File permission hardening

Firewall configuration with UFW

System log monitoring

Real-time system monitoring with htop

Cybersecurity fundamentals in system protection
